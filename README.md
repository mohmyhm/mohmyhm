### What I work on

- **LLM & Multi-Agent Security** — causal execution monitors, prompt injection defenses, trust propagation across agent pipelines
- **Trustworthy AI** — risk quantification, provenance tracking, policy enforcement for autonomous systems
- **AI for Healthcare & Science** — vision-language models, robotic assistance, clinical decision support

---

### Currently Building

🔬 **[MessageShield-MAS](https://github.com/mohmyhm/message-shield-mas)** — a causal execution monitor for multi-agent LLM pipelines. Combines noisy-OR risk propagation, Beta trust ledgers, and topology-aware placement to detect and contain prompt injection, distributed backdoors, and transitive trust attacks.
